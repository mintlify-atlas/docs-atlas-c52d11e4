    Get started
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Install cmux and start managing your AI agent workflows in minutes.
  </p>

  <Steps>
    <Step title="Install cmux">
      Download the latest release from GitHub or install via Homebrew:
      
      ```bash
      brew tap manaflow-ai/cmux
      brew install --cask cmux
      ```
      
      On first launch, macOS may ask you to confirm opening an app from an identified developer. Click **Open** to proceed.
    </Step>
    
    <Step title="Create your first workspace">
      Launch cmux and create a new workspace with `⌘ N` or from the CLI:
      
      ```bash
      cmux new-workspace --cwd ~/projects/myproject
      ```
      
      Workspaces appear as vertical tabs in the sidebar with git branch, working directory, and notification status.
    </Step>
    
    <Step title="Set up notifications">
      Wire up AI agent hooks so cmux can notify you when agents need input:
      
      ```bash
      cmux notify --title "Claude Code" --body "Waiting for input"
      ```
      
      When an agent sends a notification, its workspace tab lights up and the pane gets a blue ring. Press `⌘ Shift U` to jump to the latest unread notification.
    </Step>
  </Steps>
</div>
