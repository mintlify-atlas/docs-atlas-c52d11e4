---
title: surface.list
description: List all surfaces in a workspace
---

Lists all surfaces (panels) in the specified workspace, including their IDs, types, titles, and focus state.

## Method

```
surface.list
```

## Parameters

<ParamField path="workspace_id" type="string">
  The UUID of the workspace to list surfaces in. If not provided, uses the currently selected workspace.
</ParamField>

<ParamField path="window_id" type="string">
  The UUID of the window to target. If not provided, uses the current window.
</ParamField>

## Response

Returns an object with the following fields:

<ResponseField name="workspace_id" type="string">
  The UUID of the workspace
</ResponseField>

<ResponseField name="workspace_ref" type="string">
  The human-readable reference for the workspace (e.g., `workspace:1`)
</ResponseField>

<ResponseField name="window_id" type="string">
  The UUID of the window containing the workspace
</ResponseField>

<ResponseField name="window_ref" type="string">
  The human-readable reference for the window (e.g., `window:1`)
</ResponseField>

<ResponseField name="surfaces" type="array">
  Array of surface objects, each containing:
  - `id` (string): Surface UUID
  - `ref` (string): Human-readable reference (e.g., `surface:1`)
  - `index` (number): Position in the workspace (0-based)
  - `type` (string): Surface type (`terminal` or `browser`)
  - `title` (string): Display title of the surface
  - `focused` (boolean): Whether this surface currently has focus
  - `pane_id` (string): UUID of the containing pane
  - `pane_ref` (string): Human-readable pane reference
  - `index_in_pane` (number): Position within the pane (0-based)
  - `selected_in_pane` (boolean): Whether this is the selected surface in its pane
</ResponseField>
