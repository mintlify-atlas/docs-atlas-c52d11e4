## Overview

Creates a new pane in the specified workspace, positioned relative to the current pane. A surface is automatically created within the new pane.

## Method

```
pane.create
```

## Parameters

<ParamField path="workspace_id" type="string">
  The ID of the workspace to create the pane in. If omitted, uses the current workspace.
</ParamField>

<ParamField path="direction" type="string" default="right">
  The direction to split and create the new pane. Valid options:
  - `left` - Split to the left of the current pane
  - `right` - Split to the right of the current pane
  - `up` - Split above the current pane
  - `down` - Split below the current pane
</ParamField>

<ParamField path="type" type="string">
  The type of surface to create in the new pane. If omitted, creates a terminal surface.
  - `terminal` - Terminal surface
  - `browser` - Browser surface
</ParamField>

<ParamField path="url" type="string">
  For browser surfaces, the URL to navigate to
</ParamField>

## Response

<ResponseField name="pane_id" type="string">
  The unique identifier for the newly created pane
</ResponseField>

<ResponseField name="pane_ref" type="string">
  Human-readable reference for the pane (e.g., "p3")
</ResponseField>

<ResponseField name="surface_id" type="string">
  The ID of the surface automatically created in the pane
</ResponseField>

<ResponseField name="surface_ref" type="string">
  Human-readable reference for the surface
</ResponseField>

<ResponseField name="workspace_id" type="string">
  The ID of the workspace containing the pane
</ResponseField>

<ResponseField name="workspace_ref" type="string">
  Human-readable reference for the workspace
</ResponseField>

## CLI Usage

```bash
# Create pane to the right (default)
cmux new-pane

# Create pane in specific direction
cmux new-pane --direction left
cmux new-pane --direction up
cmux new-pane --direction down

# Create pane with browser surface
cmux new-pane --type browser --url https://example.com

# Create in specific workspace
cmux new-pane --workspace ws2 --direction right

# JSON output
cmux new-pane --json --direction left
```
