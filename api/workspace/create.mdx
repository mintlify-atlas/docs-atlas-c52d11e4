## Overview

Creates a new workspace in the specified window. If `cwd` is provided, the workspace's shell will start in that directory.

## Method

```
workspace.create
```

## Parameters

<ParamField path="window_id" type="string" optional>
  Window identifier (UUID or ref). If omitted, uses the active window.
</ParamField>

<ParamField path="cwd" type="string" optional>
  Working directory path for the new workspace. Supports:
  - Absolute paths: `/Users/name/projects/app`
  - Tilde expansion: `~/projects/app`
  - Relative paths: `./subdir` (resolved from CLI's current directory)
</ParamField>

## Path Resolution

The `cwd` parameter is resolved by the CLI before being sent to the socket:

1. **Tilde expansion**: `~/projects` → `/Users/name/projects`
2. **Relative paths**: `./app` → `{current_directory}/app`
3. **Absolute paths**: Used as-is

Source: `CLI/cmux.swift:1511`

```swift
private func resolvePath(_ path: String) -> String {
    let expanded = NSString(string: path).expandingTildeInPath
    if expanded.hasPrefix("/") { return expanded }
    let cwd = FileManager.default.currentDirectoryPath
    return (cwd as NSString).appendingPathComponent(expanded)
}
```

## Focus Behavior

The new workspace is automatically selected if the command is issued by a focus-intent operation. Socket commands issued by CLI/automation do not automatically select the workspace unless explicitly configured.

Source: `Sources/TerminalController.swift:2382`

## Example

### Create workspace in current directory

```bash
cmux new-workspace --cwd ~/projects/app
```

### Create workspace and run command

```bash
cmux new-workspace --cwd ~/projects/app --command "npm run dev"
```

Source: `CLI/cmux.swift:883`

## CLI Implementation

```bash
cmux new-workspace [--cwd <path>] [--command <text>]
```

The CLI automatically:
1. Resolves the `cwd` path
2. Creates the workspace via `workspace.create`
3. Waits 500ms for shell initialization (if `--command` is provided)
4. Sends the command via `surface.send_text`

Source: `CLI/cmux.swift:883-903`
