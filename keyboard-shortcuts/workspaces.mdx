---
title: Workspace Shortcuts
description: Keyboard shortcuts for workspace navigation and management in cmux
---

Workspaces in cmux allow you to organize multiple terminal sessions and browsers. Use these shortcuts to efficiently navigate and manage your workspaces.

## Workspace Navigation

| Shortcut | Action |
|----------|--------|
| ⌘ 1–8 | Jump to workspace 1–8 |
| ⌘ 9 | Jump to last workspace |
| ⌃ ⌘ ] | Next workspace |
| ⌃ ⌘ [ | Previous workspace |

<Note>
Use ⌘ + number keys (1-8) for instant access to your first 8 workspaces, and ⌘ 9 to quickly jump to your last workspace.
</Note>

## Workspace Management

| Shortcut | Action |
|----------|--------|
| ⌘ N | New workspace |
| ⌘ ⇧ W | Close workspace |
| ⌘ ⇧ R | Rename workspace |

## Sidebar

| Shortcut | Action |
|----------|--------|
| ⌘ B | Toggle sidebar |

The sidebar displays vertical tabs showing git branch, linked PR status/number, working directory, listening ports, and the latest notification text for each workspace.
