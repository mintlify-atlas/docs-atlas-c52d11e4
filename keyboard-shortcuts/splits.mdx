---
title: Split Pane Shortcuts
description: Keyboard shortcuts for managing split panes in cmux
---

Split panes allow you to view multiple terminals or browsers side by side in the same workspace. Use these shortcuts to create and navigate between split panes.

## Creating Splits

| Shortcut | Action |
|----------|--------|
| ⌘ D | Split right |
| ⌘ ⇧ D | Split down |

<Note>
You can split horizontally and vertically to create complex layouts with multiple terminals and browser panes.
</Note>

## Navigating Splits

| Shortcut | Action |
|----------|--------|
| ⌥ ⌘ ← → ↑ ↓ | Focus pane directionally |
| ⌘ ⇧ H | Flash focused panel |

Use ⌥ ⌘ (Option + Command) with arrow keys to move focus between panes in the direction you want to go. Press ⌘ ⇧ H to flash the currently focused panel if you lose track of which pane is active.
