---
title: Surface Shortcuts
description: Keyboard shortcuts for surface and tab navigation in cmux
---

Surfaces (also called tabs) allow you to have multiple terminal or browser sessions within a workspace. Navigate between them efficiently with these shortcuts.

## Surface Navigation

| Shortcut | Action |
|----------|--------|
| ⌘ ⇧ ] | Next surface |
| ⌘ ⇧ [ | Previous surface |
| ⌃ Tab | Next surface |
| ⌃ ⇧ Tab | Previous surface |
| ⌃ 1–8 | Jump to surface 1–8 |
| ⌃ 9 | Jump to last surface |

<Note>
Use ⌃ (Control) + number keys (1-8) to jump directly to a specific surface, and ⌃ 9 to access your last surface.
</Note>

## Surface Management

| Shortcut | Action |
|----------|--------|
| ⌘ T | New surface |
| ⌘ W | Close surface |

## Terminal Operations

| Shortcut | Action |
|----------|--------|
| ⌘ K | Clear scrollback |
| ⌘ C | Copy (with selection) |
| ⌘ V | Paste |
| ⌘ + | Increase font size |
| ⌘ - | Decrease font size |
| ⌘ 0 | Reset font size |

<Note>
Clear scrollback (⌘ K) removes all terminal history from the current surface. Font size changes apply to all surfaces in the current workspace.
</Note>

## Find

| Shortcut | Action |
|----------|--------|
| ⌘ F | Open find |
| ⌘ G | Find next |
| ⌘ ⇧ G | Find previous |
| ⌘ ⇧ F | Hide find bar |
| ⌘ E | Use selection for find |

<Tip>
Select text in your terminal and press ⌘ E to quickly search for it without opening the find bar.
</Tip>
